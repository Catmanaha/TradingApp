Trading App

This is is developed for operations with stocks
  • /stock/getall - gets all the stocks
  • /stock/ with delete http method and query string that has an id - deletes a stock
  • /stock/ with put http method and json - edits a stock
  • /stock/ with post http method and json - creates a stock
  • /stock/GetById with get http method and query string that has an id - gets a stock
  • /stock/GetByName with get http method and query string that has a name - gets all stocks with same name
